# OMNIA Lab

Website for **OMNIA** — multidisciplinary architecture & interior design practice by Dina Rosic.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Hero video

Add your autoplay loop reel to:

`public/video/hero.mp4`

Optional poster: `public/assets/hero-poster.jpg`

## Deploy (Vercel)

Push to GitHub and import the repo in [Vercel](https://vercel.com), or:

```bash
npx vercel
```

## Stack

- Next.js 16 (App Router)
- Tailwind CSS 4
- Framer Motion
